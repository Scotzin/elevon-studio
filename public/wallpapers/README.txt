WALLPAPERS DA ELEVON
====================

Coloque aqui as imagens de papel de parede (ex.: elevon-azul.jpg) e depois
cadastre cada uma em lib/wallpapers.ts:

    { file: "elevon-azul.jpg", name: "Elevon Azul", size: "1920×1080" }

Elas aparecem na ÁREA DA EQUIPE → aba "Wallpapers", com botão "Baixar".

Dicas:
- Resolução cheia (1920x1080 ou maior). Formatos: .jpg, .png ou .webp.
- Nomes de arquivo sem espaço/acento (ex.: elevon-azul.jpg, elevon-dark.png).
- Depois de adicionar os arquivos, faça commit/push (a Vercel publica no deploy).
