IMAGENS DOS NICHOS — Elevon Studio
===================================

Coloque aqui as imagens de cada nicho usadas na seção "Modelos por nicho"
da página inicial. Os nomes dos arquivos precisam ser EXATAMENTE estes
(tudo minúsculo, extensão .jpg):

  barbearia.jpg       -> Barbearias
  restaurante.jpg     -> Restaurantes e Pizzarias
  loja-roupas.jpg     -> Lojas de roupas
  estetica.jpg        -> Clínicas e Estéticas
  imobiliaria.jpg     -> Imobiliárias
  servicos.jpg        -> Prestadores de serviço

Recomendações:
- Proporção aproximada 16:10 (ex.: 800x500 px ou 1200x750 px).
- Formato .jpg (otimizado para web, idealmente abaixo de ~300 KB cada).
- Fotos nítidas e bem iluminadas, representando o ambiente/serviço do nicho.

Enquanto um arquivo não existir, o card mostra automaticamente um fundo
em gradiente azul com o ícone do nicho (fallback), então o layout NUNCA
quebra. Assim que você adicionar o arquivo com o nome correto, a foto
aparece sozinha — não é preciso mexer no código.

Para usar outra extensão (ex.: .png ou .webp) ou outro caminho, edite o
campo "image" de cada nicho em: lib/site.ts (seção 6).
